#Fundamentos do Python
Este repositório é uma coleção de algoritmos e exercícios práticos desenvolvidos para consolidar fundamentos da linguagem Python.
O conteúdo abrange desde a manipulação básica de dados até algoritmos de lógica avançada
O repositório é composto por 4 principais módulos.

##Dicionários
Foco em estrutura de chave-valor para armazenamento de dados. Realização das seguintes atividades:
Manipulação: Inserção, atualização e remoção dinâmica de itens.
Métodos: Uso de .get(), .update(), .pop() e .fromkeys().
Sistema Prático: Um sistema interativo de gestão de usuários com menu de opções.
##Funções
Modularização de código e paradigmas funcionais.
Argumentos: Implementação de parâmetros obrigatórios, opcionais, *args e kwargs.
Programação Funcional: Uso de map(), filter() e funções lambda.
Recursividade: Exemplos práticos com cálculo de Fatorial e sequências.
##Listas e Vetores
Sequências lineares e lógica de indexação
Fundamentos: Slicing, reversão de listas e List Comprehension
Algoritmos: Algoritmo de Kadane(Para encontrar a sublista de soma máxima), Permutações e Rotação.
##Matrizes
Tratamento de dados multidimensionais e álgebra linear básica.
NumPy: Criação de matrizes identidades, aleatórias e transpostas.
Operações: Soma de eixos, médias de linhas e multiplicação de matrizes
Lógica de Matriz: Identificação de simetria, manipulação de diagonais (principal e secundária) e rotação de 90°.
